# Мини-сайт (landing) для Creative Collective

Это простой статический сайт, чтобы “хоть что-то было” по адресу `http://<IP>/`.

## Как запускать

В корне репозитория (через Docker Compose / Portainer stack) поднимается сервис `site` (nginx).

- `http://<IP>/` — landing
- `http://<IP>/health` — health бекенда через прокси
- `http://<IP>/api/v1/...` — API бекенда через прокси

## Зачем это нужно

- Минимальная публичная точка входа (описание продукта, ссылки).
- Можно позже заменить на полноценный сайт/админку/веб-версию биржи.
