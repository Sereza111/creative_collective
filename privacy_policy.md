# ПОЛИТИКА КОНФИДЕНЦИАЛЬНОСТИ

## 1. Общие положения
Creative Collective ("мы", "наш") уважает вашу конфиденциальность.

## 2. Какие данные собираем:
- Email адрес
- Имя пользователя
- История заказов
- Сообщения в чатах

## 3. Как используем данные:
- Для работы платформы
- Для связи между пользователями
- Для статистики

## 4. Безопасность:
- Данные хранятся на защищенных серверах
- Используем шифрование (HTTPS)
- Пароли хешируются (bcrypt)

## 5. Ваши права:
- Удалить аккаунт
- Запросить данные
- Отозвать согласие
